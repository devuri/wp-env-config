encrypt_secret()

CryptTrait.php : 20

Encrypts the values of sensitive data in the given configuration array.

protected encrypt_secret(array<string|int, mixed> $config[, array<string|int, mixed> $secrets = [] ]) : array<string|int, mixed>

This method iterates through the provided $config array, checking each key against the list of sensitive keys provided by the optional $secrets parameter. If a key is found in the sensitive list, the value is hashed using SHA-256 before being added to the resulting $encrypted_config array. Non-sensitive values are added to the array without modification.

Parameters
$config : array<string|int, mixed>

An associative array containing keys and their corresponding values

$secrets : array<string|int, mixed> = []

An optional array of sensitive keys that need to be hashed (defaults to null)

Return values
array<string|int, mixed> —

$encrypted_config An associative array with sensitive values hashed
