public get_config_map() : array<string|int, string>

Retrieves a list of constants defined by the Setup class, but only if the WP_ENVIRONMENT_TYPE constant is set to 'development', 'debug', or 'staging'. If WP_DEBUG is not defined or is set to false, the function returns ['disabled'].

Return values
array<string|int, string> —

Returns an array containing a list of constants defined by Setup, or null if WP_DEBUG is not defined or set to false.
