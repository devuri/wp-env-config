Modifies user capabilities to allow users with 'manage_tenant' capability to install plugins.

public manage_tenant_install_plugins(array<string|int, mixed> $allcaps, array<string|int, mixed> $caps, array<string|int, mixed> $args, WP_User $user) : array<string|int, mixed>

This function checks if the user's requested capabilities include 'install_plugins' and then checks if the user has the 'manage_tenant' capability. If they do, the function allows them to install plugins by modifying the $allcaps array.

Parameters
$allcaps : array<string|int, mixed>

An associative array of all the user's capabilities.

$caps : array<string|int, mixed>

Actual capabilities being checked.

$args : array<string|int, mixed>

Adds context to the capabilities being checked, typically starting with the capability name.

$user : WP_User

The user object.

Return values
array<string|int, mixed> —

Modified array of user capabilities.
